Introduction

In this module, you will cover the following learning objectives and key activities.

Select each of the icons for more information about the learning objectives and key activities relating to this module.
  • Learning objectives
  • Key activities

At the end of this module, you will be able to:

  • identify and respond to cultural and communication barriers
  • identify various influences on communication and how misunderstandings can occur
  • identify and determine the appropriate communication technique, specifically:
    • open ended questions, affirmations, reflections and summaries
    • the difference between motivational interviewing and a coercive approach
    • the difference between collaboration and confrontation styles.
  • Watch a video about different communication techniques.
  • Consider key points about the communication techniques used in the video.
